# ウェブスクレイピングタスク

## 目的

指定されたウェブサイトから構造化データを抽出し、JSON形式で保存する。

## ターゲットサイト

- <https://www.therockmanexezone.com/wiki/Finalization>

## ページ内容・抽出内容

Black Ace set, Red Joker set

上記カテゴリのラベルの見出しとしてLV1～LV12までのテーブル表がある。
各テーブルは **5列 x 7行 = 35セル** の固定サイズで、カード名称が記載されている。

### 注意事項

- 各テーブルは必ず35枚のカードを含む
- 同じカード名が複数回出現する（重複あり）
- 重複を除去せず、テーブルの内容をそのまま保持すること
- 行順（左上から右下へ）で配列に格納する

## 出力形式

`finalization_cards.json`として以下の構造で保存

- Black Ace set → `BA`
- Red Joker set → `RJ`

```json
{
    "BA": {
        "LV1": ["<カード名1>", "<カード名2>", ... , "<カード名35>"],
        "LV2": ["<カード名1>", "<カード名2>", ... , "<カード名35>"],
        ...
        "LV12": ["<カード名1>", "<カード名2>", ... , "<カード名35>"]
    },
    "RJ": {
        "LV1": ["<カード名1>", "<カード名2>", ... , "<カード名35>"],
        "LV2": ["<カード名1>", "<カード名2>", ... , "<カード名35>"],
        ...
        "LV12": ["<カード名1>", "<カード名2>", ... , "<カード名35>"]
    }
}
```
