## 1. ノイズ（MMSF3_NOISE_OPTIONS）

Organizerでは `value`（hex文字列）をIDとして使い、`label` を表示名として使う。
BuildRecordの `mmsf3.noise` フィールドには **label（日本語名）** が入る。

| value | label |
|-------|-------|
| 00 | 初期 |
| 01 | リブラ |
| 02 | コーヴァス |
| 03 | キャンサー |
| 04 | ジェミニ |
| 05 | オヒュカス |
| 06 | キグナス |
| 07 | オックス |
| 08 | ヴァルゴ |
| 09 | クラウン |
| 0A | ウルフ |
| 0B | ブライ |

※ ブラザールーレットのノイズ（MMSF3_BROTHER_ROULETTE_NOISE_OPTIONS）は `0B`（ブライ）を除いた11個。
※ デフォルト値は `"ノーマルロックマン"` （selectboxの選択肢とは別に文字列として扱われる）

## 12. BuildRecord フィールドとマスタデータの対応

| BuildRecordフィールド | 使用するマスタ | 値の形式 |
|----------------------|---------------|---------|
| `game` | - | `"mmsf3"` 固定 |
| `version` | バージョン | `"black-ace"` or `"red-joker"` |
| `commonSections.cards[].name` | バトルカード | 日本語カード名 |
| `commonSections.abilities[].name` | アビリティ | `label` 形式（例: `"エースＰＧＭ/0"`） |
| `mmsf3.noise` | ノイズ | 日本語名（例: `"ウルフノイズ"`）※UIでは`label`を使用 |
| `mmsf3.warRockWeapon` | ウォーロック装備 | 日本語名 |
| `mmsf3.noiseCardIds[]` | ノイズドカード | hex ID（例: `"01"`） |
| `mmsf3.whiteCardSetId` | ホワイトカードセット | hex ID（例: `"00"`） |
| `mmsf3.rezonCards[]` | レゾンカード | 日本語名（例: `"バトル"`） |
| `mmsf3.brotherRouletteSlots[].noise` | ノイズ（ブライ除外） | 日本語名 |
| `mmsf3.brotherRouletteSlots[].gigaCard` | ギガカード | 日本語名 |
| `mmsf3.brotherRouletteSlots[].megaCard` | メガカード | 日本語名 |
| `mmsf3.brotherRouletteSlots[].whiteCardSetId` | ホワイトカードセット | hex ID |
| `mmsf3.brotherRouletteSlots[].rezon` | レゾンカード | 日本語名 |
