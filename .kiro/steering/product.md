# Product Overview

流星のロックマン3（Mega Man Star Force 3）のカードフォルダ（デッキ）ビューアアプリケーション。melonDS エミュレータと連携してゲーム内メモリを監視・操作し、バトルカード情報をリアルタイム表示する。

## Core Capabilities

- **カードデータベース閲覧**: ゲーム内全カードの名前・属性・攻撃力・効果テキストを検索・フィルタリング
- **ファイナライズフォルダ表示**: バージョン（BA/RJ）×レベル（1-12）ごとの変身時デッキ構成を確認
- **エミュレータ連携**: melonDS のメモリを監視し、ゲーム状態に応じた画面表示（デスクトップ版のみ）
- **多言語対応**: 日本語・英語 UI の切り替え

## Target Use Cases

- ゲームプレイ中のカードデッキ確認・戦略立案
- ファイナライズ時のフォルダ構成を事前に把握
- カード情報の検索・属性フィルタリング

## Value Proposition

- Web版: ブラウザのみでカード一覧を閲覧可能
- Desktop版: エミュレータと連携し、ゲーム進行に応じた動的表示
- 共通UIパッケージによるWeb/Desktop間の一貫した体験
